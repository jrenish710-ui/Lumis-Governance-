# 🌐 Lumis — On-Chain Governance for Web3 Communities

> *Where communities find their voice, on-chain.*

## What is Lumis?

Lumis is a mobile-first, open-source governance platform built on Solana. It lets Web3 communities — DAOs, NFT projects, DeFi protocols — propose, debate, and vote on decisions, all on-chain, in one place.

No Discord polls. No Snapshot tabs. No confusion.

Fast. Transparent. Community-owned.

## The Problem We're Solving

Web3 communities are passionate but fragmented. Governance is scattered across Discord, Snapshot, Twitter threads, and Telegram. The result: low voter participation, zero accountability, and great communities that slowly die.

Lumis fixes this by bringing everything into one on-chain home.

## Core Features (v1 Roadmap)

- [x] Project scaffolding & architecture
- [ ] Wallet connect (Phantom, Backpack)
- [ ] Create & join community spaces

- [ ] Results dashboard
- [ ] Participation points system

## Why Solana?

Voting should be free and instant. Solana makes that real — $0.00025 per transaction, sub-second finality, native wallet support.
